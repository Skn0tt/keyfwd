keyfwd

A remote keyboard forwarder, built on web technology.

Getting Started

$ npm install --global keyfwd
$ keyfwd

Welcome to keyfwd!

keyfwd allows you to control your keyboard
from any device in the world, e.g. as a remote presenter.

Open http://keyfwd.simonknott.de/odd-feather to start.
